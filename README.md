# DevBlog Engine 📝

> Engine de blog com suporte a MDX, geracao estatica e busca full-text.

## Stack
Next.js 15 • MDX • Fuse.js • RSS • ISR • Tailwind CSS

## Funcionalidades
- Postagens em MDX com preview em tempo real
- Busca full-text com Fuse.js
- RSS feed e sitemap automaticos
- Geracao de paginas estaticas (ISR)
- Tempo de build otimizado

## Metrica
Lighthouse 98/100 em performance — busca em milissegundos.
